**Ingeniero de Sistemas** y Especialista en Ciberseguridad. Mi filosofía: para defender un sistema eficazmente, debes saber cómo destruirlo.

> *"La seguridad no es un producto, es un proceso."*

- ⚔️ **Red Team (Ofensiva):** Pentesting activo, escalada de privilegios y explotación en entornos Web, Mobile, API y Active Directory.
- ⚙️ **Automatización:** Scripts tácticos en `Python`, `Bash` y `PowerShell` para DevSecOps e integración con IA.
- 📜 **Estándares:** MITRE ATT&CK, OWASP Top 10 e ISO 27001.
